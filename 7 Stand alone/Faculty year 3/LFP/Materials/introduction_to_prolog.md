# Useful commands

1. start program: **swipl**
2. consult source file:
	1. **\[filename\].**
	2. **consult(filename).**
3. start program with source file: **swipl filename.pl**
4. print working directory: **pwd.**
5. stop the program: **halt.**

# Queries

1. Use **;** to see more results after a query.
